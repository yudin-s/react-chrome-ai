Browser-side LLM React package

@yudin-s/react-chrome-ai is a React package for browser-side LLM functionality in Chrome. It targets local, browser-native AI flows rather than cloud provider APIs.

Use it for Gemini Nano, Chrome Built-in AI, LanguageModel, Prompt API streaming, structured JSON output, model readiness, model download progress, and Chrome task APIs.

Links: npm, GitHub, live example.
