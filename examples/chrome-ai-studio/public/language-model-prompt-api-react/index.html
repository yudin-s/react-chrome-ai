LanguageModel Prompt API React hooks

@yudin-s/react-chrome-ai provides React hooks for Chrome's LanguageModel Prompt API, including availability checks, session creation, monitor progress, prompt calls, streaming responses, append, clone, destroy, and context tracking.

Use it when a React UI needs complete control over browser-side LLM state: support detection, downloadable/downloading/available states, progress bars, errors, abort signals, and typed structured output.

Links: npm, GitHub, live example.
