Chrome AI model download progress in React

Chrome can download or prepare browser-managed model files when a LanguageModel session is created. React apps need to show this state clearly instead of hiding it behind a stalled button.

@yudin-s/react-chrome-ai exposes availability and progress through hooks, including useChromeAIAvailability and useChromeAISession. You can render status labels, progress bars, retry buttons, user-activation flows, and errors from React state.

Links: npm, GitHub, live example.
