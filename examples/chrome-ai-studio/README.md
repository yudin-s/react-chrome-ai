# Chrome AI Studio Example

Full React UI for the `LanguageModel` side of Chrome Built-in AI.

It demonstrates:

- support and availability checks;
- user-triggered model preparation;
- download progress;
- session ownership and teardown;
- streaming prompt output;
- context-window tracking;
- `LanguageModel.params()` display.

## Run

```bash
npm install
npm run dev
```

Open the Vite URL in a Chrome build that exposes the Prompt API.
