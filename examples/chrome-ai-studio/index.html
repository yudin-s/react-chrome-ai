React hooks for Gemini Nano and Chrome Built-in AI

@yudin-s/react-chrome-ai is a TypeScript package for React apps that use browser-native Chrome AI, Gemini Nano, the LanguageModel Prompt API, and browser-side LLM task APIs.

The hooks expose model support, availability, user activation, model download progress, session lifecycle, streaming prompts, structured JSON output, context usage, Summarizer, Translator, Language Detector, Writer, Rewriter, and Proofreader state.
